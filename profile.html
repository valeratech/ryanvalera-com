Profile

Security Operations engineer built upon a production infrastructure foundation, where Linux platforms, DNS trust, and edge security evolved into modern threat detection and defensive engineering. Investigations follow an evidence-driven methodology that correlates endpoint, network, and log telemetry to reconstruct attacker activity, validate findings, and map behavior to the MITRE ATT&CK framework.

That foundation was forged operating enterprise hosting infrastructure spanning 500+ domains, 40+ Linux servers, and high-trust production environments. Verification is treated as part of implementation, with failover, recovery, and operational changes validated before production reliance, while automation transforms repetitive administration into repeatable engineering workflows.

New technologies are approached with the same disciplined engineering methodology applied to production systems. Laboratory environments are built, instrumented, and documented until architectural behavior is understood from evidence. Whether building security laboratories, cloud infrastructure, or enterprise application platforms, every project begins with understanding the architecture, validating assumptions through evidence, and designing systems that are reliable, repeatable, and built to be maintained.
